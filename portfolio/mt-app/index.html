MT Consulting Agency App

Luglio 2018 – Settembre 2018

Collaborazione con MT Consulting Group SRLs, agenzia di comunicazione digitale con sede a Lecco che si occupa di Branding, Design, Social Media, Sviluppo web e mobile, servizi fotografici e video.

mtconsultingroup.it

Descrizione progetto

Lo scopo del progetto è stato quello di realizzare un’applicazione web per facilitare la gestione interna dei clienti e dei compiti (“tasks”) assegnati ai dipendenti dell’agenzia di comunicazione. Le tecnologie adottate per lo sviluppo dell’applicazione sono state:

Back End: Java
Database: MySQL
Front End: Angular 6 + Bootstrap 4

Funzionalità principali
  • Tasks

    Lista dei compiti assegnati, divisi per giorno. Per ogni task sono associate le seguenti informazioni: titolo, descrizione, cliente, utenti assegnati, completato (si/no), data di scadenza e tempo stimato. Una volta finito il task, l’utente può completarlo tramite uno switch. Inoltre, è possibile filtrare i task per utente, cliente oppure data.

  • Today

    Una dashboard dove poter visualizzare i tasks assegnati all’utente loggato per quel giorno. Una barra di avanzamento che si aggiorna dinamicamente mostra la percentuale dei tasks completati.

  • Clienti

    Lista dei clienti dell’agenzia. Ogni cliente ha una scheda dove è possibile visualizzare le sue informazioni, i tasks completati o non completati, le mansioni associate a quel cliente ed eventuali documenti caricati. Tramite una barra di ricerca e dei filtri è possibile selezionare uno o più clienti a seconda delle esigenze

  • Utenti

    Lista dei dipendenti dell’agenzia. Ogni utente ha una scheda dove è possibile visualizzarne le informazioni, i tasks completati o non completati e le informazioni di contatto.
